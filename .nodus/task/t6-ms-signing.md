# t6 — Подпись драйвера для пользователей

**Статус:** 🟡 стратегия выбрана (бесплатный OSS-путь); активные шаги — после стабилизации драйвера (t2–t3)
**Ждёт:** ничего блокирующего; подача в фонд — когда драйвер реально работает
**Кто делает:** я готовлю репозиторий/заявку, ты — финальное решение об open-source

## Суть простыми словами
Сейчас драйвер подписан тестовым сертификатом — ставится только на машину с Test
Mode, для пользователей это неприемлемо. Классический путь (EV-сертификат
~$300–500/год + Microsoft Partner Center) нам не подходит по деньгам, и западные
CA не выдают сертификаты в РФ.

**Выбранная стратегия (11.06.2026): бесплатный путь для open-source.**
Фонды SignPath Foundation (signpath.org) и OSSign (ossign.org) бесплатно
подписывают open-source проекты, включая kernel-драйверы с attestation-подписью
Microsoft, своим сертификатом и своим аккаунтом — нам не нужны ни EV, ни юрлицо.
Драйвер и так производная MIT-кода SYSVAD: открываем ТОЛЬКО драйвер (отдельный
публичный репозиторий). Сам Nodus закрыт лишь на этапе разработки; распространяться
будет бесплатно (добровольные пожертвования) — для фонда подписи это неважно,
их условия касаются только подписываемого драйвера.

**Параллельный план Б на время до подписи:** MVP распространяется без своего
драйвера — пользователь сам ставит бесплатный VB-Audio Virtual Cable (не вкладывать
в инсталлятор — это платная лицензия!), Nodus его уже детектит (fallback готов).

## Шаги
- [ ] (после t2–t3) Вынести драйвер в отдельный публичный репозиторий, MIT,
      с CI-сборкой — фонды требуют сборку из публичного репо
- [ ] Проверить соответствие условиям (signpath.org/terms.html): OSI-лицензия,
      нет проприетарного кода, активная поддержка, документированная функциональность
- [ ] Подать заявку в SignPath Foundation (и/или OSSign), встроить их подпись в CI
- [ ] Обновить install.ps1/инсталлятор: убрать Test Mode и импорт тестового сертификата
- [ ] В Nodus: кнопка/инструкция «установить VB-Cable» как временный путь (план Б)

## Заметки
- Azure Trusted Signing ($9.99/мес) kernel-драйверы НЕ подписывает — не вариант.
- «Серые»/перепроданные сертификаты — отзыв подписи и пометка малварью; запрещено.
- Запасной вариант: партнёрство — подпись через Partner Center знакомой компании.
- CI уже подписывает пакет тестово; точка интеграции фонда — тот же шаг подписи.
