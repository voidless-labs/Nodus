# Nodus

**Нодовый виртуальный аудио-роутер для Windows.** Холст с нодами вместо
табличного микшера — соединяешь источники звука (приложения, микрофон, систему)
с выходами (наушники, OBS, виртуальный микрофон) проводами и управляешь
громкостью, mute, балансом и эффектами **на каждом маршруте**.

> 🌐 [English version](README.md) · 🆓 Nodus бесплатен (только пожертвования)
> · 🛠️ В активной разработке

---

## Что умеет

- **Визуальный роутинг** — перетаскиваешь источники и выходы на холст и тянешь
  между ними провода. Никаких фиксированных строк микшера.
- **Контроль на маршруте** — громкость, mute и баланс живут на каждом проводе, а
  не только на источнике. Один источник может идти в несколько выходов, у каждого
  свои настройки.
- **Mixer и Splitter** — много входов в один (Mixer) или один вход на много
  выходов (Splitter).
- **Виртуальные устройства** — виртуальные микрофоны/выходы Nodus, которые другие
  приложения (Discord, OBS, игры) видят как реальные устройства.
- **Автоопределение приложений** — запущенные аудио-приложения (Spotify, Discord,
  Chrome, игры…) появляются автоматически, со своими настоящими иконками.
- **Сцены** — несколько схем роутинга с переключением между ними.

### Пример

```
Игра ─┬─→ Наушники             (слышишь ты)
      └─→ Nodus Virtual Out     (слышит OBS)

Микрофон ───┐
Spotify ────┴─→ Mixer ─→ Nodus Virtual Mic ─→ Discord
```

## Состояние

MVP аудио-движка работает на реальном железе (захват/вывод через WASAPI,
громкость/mute/pan на маршруте, splitter, mixer, изоляция приложений через
process loopback). Виртуальные микрофоны/выходы маршрутизируются через VB-Audio
или kernel-драйвер Nodus, если он установлен.

В работе (Phase 2): динамическое создание виртуальных устройств из UI
(kernel-драйвер), реальный DSP у эффектов (сейчас passthrough), logic/trigger
ноды и push-to-talk.

## Стек

- **UI** — Tauri 1.x · React 18 · TypeScript · Vite
- **Бэкенд** — Rust · WASAPI / Windows Core Audio
- **Виртуальные устройства** — kernel-аудиодрайвер Windows (на базе SYSVAD)

## Разработка

> Требуется Windows, Node.js и toolchain Rust.

```bash
npm install
npm run dev          # только UI, в браузере (http://localhost:1420)
npm run tauri dev    # полное десктоп-приложение (реальный аудио-бэкенд)
npm run build        # production-сборка UI
```

Rust-бэкенд:

```bash
cd src-tauri
cargo check
cargo test
```
